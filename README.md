# MINDPULSE  
### Lightweight Personal Productivity & Mental Tracking System

> A fast, privacy-focused client-side web application designed to help users track productivity, behavior patterns, focus levels, and mental state through simple daily interaction and visual analytics.

---

## Overview

**MindPulse** is a lightweight web-based system built to help users understand and improve their productivity, focus, and behavioral patterns.

Unlike heavy productivity tools, MindPulse is designed to be:
- Fast and minimal  
- Fully client-side (no complex backend)  
- Privacy-focused  
- Easy to deploy and use anywhere  

---

## Core Objectives

- Improve daily productivity awareness  
- Track focus, mood, and behavior patterns  
- Provide simple but powerful analytics  
- Encourage self-improvement through visualization  
- Maintain full user privacy (local-first design)  

---

## System Concept

MindPulse operates as a **personal self-tracking dashboard**:

- Users log daily focus, mood, and productivity levels  
- Data is visualized into patterns and trends  
- System highlights improvements or declines  
- Helps users identify habits affecting performance  

---

## Key Features

### Daily Tracking System
- Mood logging  
- Focus level tracking  
- Productivity score input  
- Habit tracking (optional)  

---

### Analytics Dashboard
- Weekly and monthly trend graphs  
- Productivity pattern visualization  
- Focus vs distraction comparison

---

### Insight System
- Detects low-performance days  
- Identifies productivity cycles  
- Shows improvement trends over time  

---

### Minimal & Fast UI

- Mobile-friendly layout

## Tech Stack

### Core Technologies
- HTML — Structure of the application

---

### Data Handling
- LocalStorage — Stores user data directly in the browser

---

### Visualization
- Chart.js — Used for graphs and analytics dashboards
