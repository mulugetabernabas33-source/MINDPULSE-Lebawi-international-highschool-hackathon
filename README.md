# MINDPULSE-Lebawi-international-highschool-hackathon
MindPulse is a behavioral intelligence web application designed to help users improve their mental wellbeing by analyzing daily habits such as sleep, stress, mood, and activity. Unlike traditional health trackers, MindPulse goes beyond data collection by detecting patterns, generating meaningful insights, and providing personalized recommendations that adapt over time.

Built with a lightweight, client-side architecture, the system uses local data and smart logic to predict burnout risk early and guide users toward healthier routines. It also includes weekly, monthly, and yearly reports, offering a long-term view of behavioral trends.

MindPulse is especially relevant in environments like Ethiopia, where awareness of mental health patterns is still growing, and many students and young professionals face high stress, irregular sleep, and digital overload. This project aims to provide an accessible, practical tool for understanding and improving daily wellbeing.
